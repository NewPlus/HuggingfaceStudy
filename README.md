## Contents Code
- Contents Number(2) / Current Process(3)
    - Contents Number : 2자리, 각 내용별 순서대로 코드 부여
    - Current Process : 3자리, 커밋 당시 진행 상태대로 코드 부여, 코드숫자(n)%를 표현, 090은 90% 의미
    - ex) 01056 : 1번째 내용의 56% 진행된 파일을 커밋
